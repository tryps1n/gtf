# GTF - Google Translate CLI

A simple command line translating tool which can translate phrases to English. Default language is German.

## Quick Install

### Method 1
```bash
# Clone
git clone https://github.com/tryps1n/gtf
cd gtf-translator
```

### Method 2
```bash
# Clone
git clone https://github.com/YOURNAME/gtf-translator.git
cd gtf-translator

# Build
mkdir build && cd build
cmake .. && make
```

Run with ./gtf afterwards, or use with command by running

```bash
sudo cp gtf /usr/local/bin/
``` 

## Usage
```bash
gtf hello world
```
## Dependencies 
- CMake
- Curl
